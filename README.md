# NFL Analytics — nflfastR Breakdowns

A collection of NFL team and player analysis projects built in R using the nflverse ecosystem.
Each breakdown explores offensive and defensive analysis using play-by-play participation data.

## Author
**Manny Tovar**

## About
These projects use play-by-play and participation tracking data from nflfastR to break down
NFL teams schematically. Looking at personnel groupings, coverage tendencies, efficiency
metrics, and situational splits. All analysis is reproducible from the source scripts.

## Data Source
All data is sourced from the [nflverse](https://nflverse.nflverse.com)
